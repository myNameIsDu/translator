## Install

```shell
pnpm i translator-server -d

pnpm translator
```

然后打开 `http://localhost:7733/`

## 约定

1. 工作目录下 `src/locales/complete.json` 为存放语言包的文件
2. 所有需要翻译的文案及使用当前工具的工作目录为`src`，即扫描收集文案会在 `src` 目录下进行

## 开发

1. 执行 `watch`

```
p dev-cli
```

2. 当前包 link 到全局

```shell
p  link --global
```

3. 在需要有国际化需求的项目内执行

```shell
translator dev
```
